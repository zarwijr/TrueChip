Chip đã xác thực

QUY TRÌNH

0x01

Đọc UID

Đầu đọc gửi lệnh GET_ID, chip FPGA trả về UID 16 byte gắn cố định trong phần cứng.

0x02

Thử thách ngẫu nhiên

Server sinh nonce 16 byte ngẫu nhiên, gửi xuống chip qua đầu đọc để yêu cầu chứng minh khóa bí mật.

AES

Đối chiếu server

Server tự tính AES-128(khóa bí mật, nonce ⊕ UID) và so khớp hằng-thời-gian với phản hồi từ chip.
